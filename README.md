# AI-Powered Smart Grid Intelligence Platform

## Overview

AI-Powered Smart Grid Intelligence Platform is a web-based application that helps monitor, analyze, and optimize smart grid operations using Artificial Intelligence and Machine Learning techniques.

The platform provides insights into energy consumption, demand forecasting, anomaly detection, and grid performance to improve efficiency and reliability.

## Features

* Real-time energy monitoring
* AI-based demand forecasting
* Smart analytics dashboard
* Data visualization and reporting
* Grid performance monitoring
* User-friendly interface

## Tech Stack

* Frontend: React.js, Vite
* Backend: Node.js
* Database: MongoDB
* AI/ML: Python, Machine Learning Models

## Installation

1. Clone the repository

```bash
git clone https://github.com/satyam8736/AI-Powered-Smart-Grid-Intelligence-Platform.git
```

2. Navigate to the project folder

```bash
cd AI-Powered-Smart-Grid-Intelligence-Platform
```

3. Install dependencies

```bash
npm install
```

4. Run the project

```bash
npm run dev
```
